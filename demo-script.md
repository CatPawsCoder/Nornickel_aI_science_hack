# Сценарий видео-демо (≤5 минут)

Записывайте экран браузера с `http://127.0.0.1:8017` (сервер: `python -m uvicorn backend.app:app --port 8017` из корня проекта). Можно с закадровым комментарием или без — таймкоды ориентировочные.

## 0:00–0:30 — Заголовок и статистика
- Открыть страницу. Показать шапку: «101 публикация · 20 843 условия · 233 утверждения · 74 сущности · LLM: online».
- Одна фраза: «Это карта знаний R&D для горно-металлургической отрасли — граф, где числа физически не могут быть придуманы моделью».
- Показать стартовый граф (топ процессов и материалов по числу упоминаний).

## 0:30–1:30 — Запрос 1: обессоливание воды (числа + верификация)
- Кликнуть пример «Какие методы обессоливания воды подходят...».
- Дождаться ответа (~2 сек). Прокомментировать:
  - блок «Как система поняла запрос» — сущности и числовые ограничения извлечены **до** обращения к LLM;
  - блок «Числовые данные из источников» — таблица с дословной цитатой и источником;
  - если есть — блок «Зоны разногласий».
- Кликнуть по узлу-условию на графе справа → показать всплывающую карточку с цитатой.

## 1:30–2:30 — Запрос 2: циркуляция католита (мировая практика)
- Кликнуть пример про электроэкстракцию никеля.
- Прокомментировать раздел «Выводы из базы знаний»: каждое утверждение — с достоверностью (🟢/🟡/🔴), гео-меткой (🇷🇺/🌍) и ссылкой на узел/источник.
- Показать фильтр «география — зарубежная» — переключить и повторить запрос, чтобы показать фильтрацию.

## 2:30–3:30 — Запрос 3: Au/Ag/МПГ штейн-шлак (сравнение источников)
- Кликнуть пример про распределение благородных металлов.
- Показать таблицу источников с годами и гео-меткой.
- Кликнуть узел-публикацию на графе → карточка документа с claims.

## 3:30–4:15 — Запрос 4: закачка шахтных вод (пробелы в знаниях)
- Кликнуть пример про закачку шахтных вод.
- Если в ответе есть блок «Пробелы в знаниях» — показать его: система честно говорит, где данных нет, а не выдумывает.

## 4:15–5:00 — Итог
- Кратко: гибридная архитектура (граф = источник истины, LLM = интерфейс), готовность к масштабированию на новые домены, дашборд `/api/stats` для покрытия по направлениям.
- Финальный кадр — общий вид графа знаний.

## Технически перед записью
```bash
cd nauchny-klubok
python -m uvicorn backend.app:app --host 127.0.0.1 --port 8017
# открыть http://127.0.0.1:8017 в браузере, окно ~1400x900
```
