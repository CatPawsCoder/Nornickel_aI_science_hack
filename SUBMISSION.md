## ⚡ Развёртывание для жюри (Docker, CPU-only, влезает в 12 ядер / 32 ГБ / без GPU)

GPU нужен только на этапе сборки корпуса (OCR) — **рантайм полностью CPU**.
Жюри не пересобирает корпус: предсобранные данные (граф Kùzu + FTS5-индекс, 274 МБ)
скачиваются контейнером при первом старте с публичной ссылки (уже выложены):

```bash
git clone https://github.com/CatPawsCoder/Nornickel_aI_science_hack.git
cd Nornickel_aI_science_hack
DATA_BUNDLE_URL="https://huggingface.co/datasets/catpawws/nornickel-hack-demo/resolve/main/klubok-data-v4.tar.gz" \
DATA_BUNDLE_SHA256="ddbfd805e546c055097e46480b426827a00690bf2700ea7071100297252c8c81" \
docker compose up
# открыть http://localhost:8017
```
SHA256 бандла проверяется контейнером перед распаковкой (защита от подмены архива);
симлинки и абсолютные пути внутри tar отклоняются.
LLM-ключи опциональны: без них система работает в граф-режиме
(детерминированный поиск + структурированный ответ; теряется только LLM-«полировка»).

Проверено: все рантайм-зависимости (`requirements-docker.txt`) — готовые wheels,
устанавливаются в чистое окружение без компилятора; импорт всех модулей проверен.

## Что нужно сделать вручную (требует ваших аккаунтов)

### 1. Ссылка на исходный код в VCS
Локальный git-репозиторий уже готов (`git log` — 2+ коммита). Нужно создать репозиторий на GitHub/GitLab и запушить:
```bash
cd nauchny-klubok
git remote add origin <URL_вашего_репозитория>
git push -u origin master
```
⚠️ Перед пушем убедитесь, что `.env` не в репозитории (проверьте `git ls-files | grep .env` — должно быть пусто; `.gitignore` уже это гарантирует).

### 2. Ключи — перевыпустить после хакатона
В процессе разработки в чат были вставлены в открытом виде: Yandex AI Studio API-ключ, OpenRouter API-ключ, GigaChat OAuth-credentials. Рекомендуется перевыпустить все три после сдачи (особенно OpenRouter — привязан к оплате).

### 3. Запись видео-демо
Следуйте `demo-script.md`. Запустите сервер (`python -m uvicorn backend.app:app --port 8017`), откройте `http://127.0.0.1:8017`, запишите экран (Windows: Win+Alt+R, или OBS Studio). ≤5 минут.

### 4. Облачный диск (Яндекс/Google)
Загрузить:
- архив исходного кода (`git archive` или zip папки проекта без `.env`, `data/raw`, `data/kg.kuzu`)
- видео-демо
```bash
cd nauchny-klubok
git archive -o ../nauchny-klubok-src.zip HEAD
```

### 5. Презентация
**`presentation/nauchny-klubok-NN.pptx`** — на официальном шаблоне «Норникель 2026» (основная).
`presentation/nauchny-klubok.pptx` — авторский дизайн (запасная).
Загрузить на диск или в отдельную ссылку.

### 6. Развёрнутое решение (опционально)
Локальный FastAPI-сервер можно развернуть на Render/Railway/VPS для публичной ссылки жюри — не обязательно, но плюс к оценке. Требует отдельного шага деплоя (не сделано в рамках этой сессии).
