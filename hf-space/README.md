# 🧶 Научный клубок — верифицируемая карта знаний R&D

Демо решения трека «Научный клубок» (Норникель, хакатон 2026).

- 2 090 документов корпуса кейса · 292 155 верифицированных числовых условий ·
  665 LLM-утверждений с дословными цитатами · граф Kùzu (Cypher)
- Числа извлекаются детерминированной грамматикой и проверяются string-match
  к источнику — галлюцинация не может попасть в граф
- Запросы на естественном языке: числовые ограничения, география, годы
